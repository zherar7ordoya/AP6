Projects structure:

- ASP.NET MVC Store.csproj - Model and application logic

- \Tests\Tests.sln - Tests for application logic

- \Presentation - Presentation classes (Web)

An article covering this example is available at:

	Documentation\mvcsharp.org\Reworking_ASPNET_MVC_Store\Default.html

and online:

	http://www.mvcsharp.org/Reworking_ASPNET_MVC_Store/Default.aspx

To build and run the example open the solution file in the VS IDE
or run Build.bat (it uses MSBuild tool - a part of the .NET framework,
so ensure it is accessible through the PATH environment variable).
